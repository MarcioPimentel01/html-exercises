HTML Basics - Add Audios

Please see example below

Girl On Top - Amy Lynn & the Honeymen

Hotlanta - Track Tribe

Lazy Laura - Quincas Moreira

Please see HTML code below

<p>Hotlanta - Track Tribe</p>

<audio controls loop>

<source src="audios\Girl On Top - Amy Lynn & the Honeymen.mp3">

<source src="audios\Girl On Top - Amy Lynn & the Honeymen.wave">

</audio>


<p>Hotlanta - Track Tribe</p>

<audio controls loop>

<source src="audios\Hotlanta - Track Tribe.mp3">

<source src="audios\Hotlanta - Track Tribe.wave">

</audio>


<p>Lazy Laura - Quincas Moreira</p>

<audio controls loop>

<source src="audios\Lazy Laura - Quincas Moreira.mp3">

<source src="audios\Lazy Laura - Quincas Moreira.wav">

</audio>


Please see explanation below

<audio controls loop>:

The <audio> element is used to embed audio content.

• controls: Adds audio controls (play, pause, volume, etc.) to the audio player.
• loop: Specifies that the audio should loop when it reaches the end.

Audio Sources <source>

The <source> elements provide alternative audio files in different formats (mp3 and wav).
The browser will choose the first compatible source it supports.
The “src” attribute specifies the file path for each audio file.

Text Content <p>:

Paragraphs <p> with text are included between the audio elements, providing a description or title for each audio track.

In summary, this code embeds three audio players into the web page, each with a set of controls and sources for different audio formats. The paragraphs below each audio player provide a title or description for each audio track. Users can play, pause, adjust volume, and loop the audio tracks using the provided controls.
