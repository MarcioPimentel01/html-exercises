<hr>

The <hr> element is used to represent a thematic break or a horizontal line in the content of a web page. It is a self-closing tag, meaning it doesn't have a closing tag.

<p> Paragraph

In HTML, <p> stands for "paragraph." The <p> element is used to define a paragraph of text on a web page. It is a block-level element, meaning it creates a new block or section on the page.

<pre> Preformatted text

                     In HTML, The <pre> element is used to define preformatted text, 
                     where both spaces and line breaks are displayed exactly as they appear in 
                     the HTML code. It is a block-level element, and its primary purpose is to 
                     preserve both whitespace and line breaks.

<a> Anchor

The <a> element is used to create hyperlinks, allowing you to link to other web pages, resources, or locations within the same document.
The href attribute in the <a> element specifies the URL or destination of the link.

Please see some examples:

Click me to a external link. 
For the external link above We used the following HTML code.

                    <a href="https://google.com" target="_blank" ,title="This external link goes to Google">
                        Click me to a external link.
                        </a>
                    
                    
As you already knows the href attribute in the <a> element specifies the URL or destination of the link.

<img> Add Images

The <img> tag in HTML is used to embed images in a web page.
Here's a basic example of how the <img> tag is typically used:

Golden Retriver

The Golden Retriver is a very good dog.

HTML IMAGE CODE

                <h2>Golden Retriver</h2>    
                <img src="images\golden-retriever-puppy.jpg", 
                height="300",
                alt="This is a picture of a Golden Retriver">
                <h4>The Golden Retriver is a very good dog.</h4>

Gif example

Gif of a joyful dog!



Gif example

            <h2>Golden Retriver</h2>    
            <img src="images/dog_gif.gif", 
            height="300",
            alt="This is a picture of a Golden Retriver">
            <h4>Gif of a joyful dog!.</h4>

Explanation of the attributes:
src: This attribute specifies the source (URL or file path) of the image. In the example, it's set to "image.jpg,"
but you would replace it with the actual path or URL of your image.

alt: This attribute provides alternative text for the image. It is displayed if the image cannot be loaded or is read
by screen readers for accessibility. It's considered good practice to include descriptive alt text.
