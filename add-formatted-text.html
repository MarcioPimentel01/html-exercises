Please see the code below

 <p>This is normal text</p>
 <p>This is <b>bold</b> text</p>
 <p>This is <i>italic</i> text</p>
 <p>This is <u>ubderline</u> text</p>
 <p>This is <del>deleted</del> text</p>
 <p>This is <big>big</big> text</p>
 <p>This is <small>small</small> text</p>
 <p>This is <sub>subscript</sub> text</p>
 <p>This is <sup>superscript</sup> text</p>
 <p>This is <tt>monospaced</tt> text</p>
 <p>This is <mark>marked</mark> text</p>
 <p>This is <mark style="background-color:aquamarine">marked choosing color - aquamarine</mark> text</p>

Please see the explanation below

  1. Text Formatting Tags:
  • <b>: Makes text bold.
  • <i>: Makes text italic.
  • <u>: Underlines text.
  • <del>: Displays text as deleted, often with a strikethrough line.
  • <big>: Makes text larger than the surrounding text.
  • <small>: Makes text smaller than the surrounding text.
  • <sub>: Creates subscript text, often used for chemical formulas or footnotes.
  • <sup>: Creates superscript text, commonly used for exponents or footnotes.
  • <tt>: Displays text in a monospaced font, often used for code snippets.
  • <mark>: Highlights text as "marked," typically rendered with a yellow background.
  1. Style Attribute:
  • The last line demonstrates how to customize the appearance of the <mark> tag using the style attribute. In this case, it sets the background color to "aquamarine."

In Summary:

  • This HTML code showcases various text formatting options. Paragraphs are defined using <p> tags. Individual elements within these paragraphs are formatted using specific tags like <b> for bold, <i> for italic, and <sub> for subscript. Additionally, the <mark> tag with the style attribute demonstrates how to customize the highlighted text's appearance, in this case, setting the background color to aquamarine.
