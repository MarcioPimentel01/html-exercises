HTML Basics - Add Videos

Please see example below

Please see code below

        <h3>&<video>How to add video using HTML</h3>
        <video controls autoplay muted loop height="300">
        <source src="images\Videos\ABIGAIL ARCANE - DC COMICS ENCYCLOPEDIA - [TWO SAMURAIS].mp4">
        </video>

Please see explanation below

        <video>: This is the HTML5 video element, used to embed video content on a webpage.
        
        controls: This attribute adds video playback controls (play, pause, volume, etc.) to the video player.
        
        autoplay: This attribute automatically starts playing the video as soon as it's loaded.
        
        muted: This attribute mutes the audio of the video. In this case, the video will play without sound.
        
        loop: This attribute makes the video restart from the beginning when it reaches the end, creating a continuous loop.
        
        height="300": This attribute sets the height of the video player to 300 pixels.
        
        <source>: This is a child element of the <video> element used to specify the video source. It allows for multiple video formats to be provided, and the browser will choose the first supported format.
        
        src="images\Videos\ABIGAIL ARCANE - DC COMICS ENCYCLOPEDIA - [TWO SAMURAIS].mp4": This attribute specifies the path to the video file. In this case, the video file is named "ABIGAIL ARCANE - DC COMICS ENCYCLOPEDIA - [TWO SAMURAIS].mp4" and is located within the "images/Videos/" directory relative to the HTML file.
